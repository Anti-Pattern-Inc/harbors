表参道駅から徒歩3分の
貸し会議室

HarborS 表参道は港区青山に位置する貸し会議室です。表参道駅A4出口から徒歩3分という駅近で、ご利用しやすい好立地にあります。
会議室内には無料Wi-Fiや大画面モニターも完備しており、イベントやセミナーなど幅広い用途でご利用いただけます。また、検温器をはじめアルコール消毒液、換気窓などコロナ対策も徹底しております。
ご利用の際は会議室予約フォームよりご予約ください。

目的・用途

HarborS 表参道の貸し会議室は表参道駅の程近くにも関わらず静かな通りに位置しています。
会議や打合せ、商談はもちろん、企業の研修や試験、面接などビジネスのあらゆるシーンでご活用いただけます。
また、個人の方でも利用しやすい価格のため、セミナー会場や勉強会にもご使用いただいております。
ご利用規模に合わせて収容人数も柔軟に対応可能ですので、ぜひご活用ください。

ACCESS

HarborS 表参道は、港区青山エリアに位置しており、表参道駅A4出口から徒歩3分という好立地です。
住所は「港区南青山」とブランド力抜群のこのエリアは、渋谷、原宿、麻布、赤坂、六本木エリアとも隣接しており、どこからのアクセスも抜群の立地という特徴があります。

[周辺からのアクセス]

六本木：千代田線(乃木坂駅利用)で2分 (徒歩25分圏内)

赤坂：千代田線で6分 (徒歩30分圏内)
