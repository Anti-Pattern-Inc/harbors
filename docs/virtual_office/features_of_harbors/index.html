港区南青山のバーチャルオフィス
HarborS表参道の特長

HarborS表参道のバーチャルオフィスに興味を持っていただき、ありがとうございます。このページでは弊社が提供するサービスやその魅力について、手前味噌ではありますがご紹介させていただきます。
今後、ビジネスを始められる方やバーチャルオフィスの利用を検討していらっしゃる方のお役に立てれば幸いです。

HarborS表参道について

1．エンジニアの会社が運営

HarborS表参道は株式会社アンチパターンが運営しています。アンチパターンは「日本のソフトウェアエンジニアを憧れの職業へ」というミッションを掲げ事業運営しています。
エンジニア向けのサービスを主に提供しておりますが、バーチャルオフィスはエンジニア以外のお客様にもご利用いただけます。

2．エンジニアとビジネスオーナーの出会いの場に

ではなぜ、エンジニア向けのサービスを提供する会社がバーチャルオフィスを運営するのかというと、エンジニアとビジネスオーナーの架け橋になりたいという思いがあるからです。
バーチャルオフィスを利用するのは設立間もないスタートアップであったり、オフィスを必要としない業種などが多く、エンジニアとの親和性が高いと考えられます。HarborS表参道がそういった方々を繋ぎ、出会いの場となることを目指しております。

3．ゆくゆくは拠点展開でさらに使いやすく

現在は表参道の1拠点のみですが、ゆくゆくは拠点展開を目指しています。拠点が増えることで例えば、登記などで利用する住所と郵便物の受け取り場所を選べたり、会議室や打ち合わせなどの選択肢が増えるなど、よりお客様にとって利便性の高いサービスの提供ができるのではないかと考えております。

HarborS表参道について簡単にですが知ってもらえたところで、次は弊社の提供するバーチャルオフィスの魅力についてお話ししたいと思います。

HarborS表参道の魅力①
立地

まずは立地についてです。バーチャルオフィスの立地選びの良し悪しは大きく二つの基準で決まります。
一つは利便性、もう一つが土地の持つブランドやイメージです。

1．南青山のブランド力

HarborS表参道は「東京都港区南青山」の住所をお貸しすることができます。
バーチャルオフィスを利用する際には、その土地の持つイメージも大変重要な要素になってくるので事業内容や顧客の層にマッチした場所を利用したいと考える方は多く、弊社にも住所にこだわってバーチャルオフィスを探されているお客様が沢山いらっしゃいます。

弊社の提供する「港区南青山」という住所を聞いてどのようなイメージを抱くでしょうか？おそらく多くの人は高級感がありお洒落な街という想像をするでしょう。それは事業に使った場合も同様で、都内の一等地に当たる「南青山」の住所を利用することで、その高級感や洗練されたイメージから事業への安心感が生まれ、信頼感を高めることができます。
また、アパレルなどの業種においてはお洒落な街というイメージが顧客の期待感を高めることにもつながります。

弊社の提供する「港区南青山」の住所は、どんな業種にも馴染みやすく、高いブランド力を兼ね備えた非常に使い勝手の良い住所であると自負しております。

2．駅が近く使いやすい

表参道駅から徒歩3分というのも是非知っていただきたいポイントです。駅から近ければ、郵便物を取りに来る際やカフェスペースを使った打ち合わせの際にも利用しやすく、時間と手間を軽減することができます。

前述のようなブランド力と駅からの近さを兼ね備えたバーチャルオフィスとなると、その数はかなり限られてきます。

3．周辺施設の充実度

HarborS表参道の周辺にはカフェや食事処が豊富で、少し足を伸ばせば表参道ヒルズがあります。
わざわざバーチャルオフィスに出向くのではなく、日常の買い物やランチのついでに利用できるのも魅力の一つだと考えております。

HarborS表参道の魅力②
サービスの種類

まずは立地についてご紹介しましたが、例え場所が良くてもバーチャルオフィスの機能やサービスが不十分だと、使い勝手は悪くなってしまうので、オプションの選択肢や自由度も重要なポイントになります。
バーチャルオフィスのサービス提供体制によっては、基本料金に不要なサービスが含まれ、その分無駄に料金を払ってしまうということもあります。

弊社ではそういったミスマッチを防ぐため、基本料金で提供するサービスをシンプルなものにし、お客様自信が必要なオプションを選び、ご利用いただけるようにサービス設計をしております。

ここではオプションサービスについてご説明させていただきます。

1．24時間使える専用ポスト

バーチャルオフィスは住所の利用がメインの用途になるため、当然会社や利用者に向けた郵便物が届きます。弊社ではオプションサービスとしてお客様ごとの専用ポストの提供を行なっております。専用ポストは24時間365日ご利用いただけるので、いつでも自由に郵便の取り出しができます。
また、ポストに入りきらない荷物は受付でお預かりも可能です。書留などの場合は代理で受領し、専用ポストに投函させていただくので再配達などの手間がかかりません。

2．遠くにお住まいの方でも安心の郵便転送

遠方で事業を行っていても都内の住所を利用したい、という場合もあると思います。しかし、郵便物などを引き取りに行くだけで時間も交通費もかかってしまうので、バーチャルオフィスの利用は難しいと考えている方もいらっしゃるかもしれません。
弊社ではそういった方のためにオプションで郵便物の転送サービスをご用意しております。毎週1回郵便物がまとめて郵送されてくるので、ご自身で引き取りに来る必要が無く遠方でも安心してご利用いただけます。
オンラインでの説明会も行なっておりますので、契約のためにわざわざ足を運ぶ必要もございません。

郵便転送サービスは遠方にお住まいの方に限らず、多忙で郵便物を受け取りにいらっしゃる時間が無い方などにもおすすめのサービスとなっております。

3．セキュリティサービスにもなる電話転送

バーチャルオフィスを利用した場合に、名刺やHPに掲載されている住所は港区なのに、電話番号の市外局番が東京03じゃないと違和感があるのではないかと不安な方もいらっしゃるでしょう。そのような場合にはお客様ごとに東京03の番号を付与し、お持ちの携帯電話に転送するオプションサービスもございます。
また、セキュリティの観点から自宅や個人の携帯電話の番号を載せるのに抵抗がある方でも安心して利用できる電話番号を提供することができます。
名刺などに記載されている代表番号が携帯電話や見覚えのない市外局番だと、相手方が不安を覚える場合もあります。携帯があれば仕事ができる世の中ですが、ビジネスにおける信用を確保するという用途にも電話転送サービスは活用することが可能です。

4．打ち合わせに使えるテラス&カフェ

バーチャルオフィスを利用するお客様はご自宅や作業場での仕事が多いのではないでしょうか。オフィスを持たないワークスタイルの場合、急な来客や打ち合わせなどが入った際に利用できるスペースが必要になります。
弊社ではカフェスペースだけでなく、屋上のテラスもバーチャルオフィス会員様に開放しており、予約不要でご利用いただけます。
打ち合わせなどの用途以外にも、日々の作業の気分転換としてカフェやテラスのご利用も可能です。

5．大人数用会議室もあり

カフェやテラスなどのオープンなスペースでは話しづらい場合や大人数での会議を行う場合には、会議室もご利用いただくことが可能です。人数や用途に合わせて二つの会議室を繋げての利用も可能で最大20人まで収容可能です。

会議室のご利用には事前に予約が必要です。予約に際してはキャンセル料がかからない仮予約期間を設けておりますので、気軽にご予約いただけます。

HarborS表参道の魅力③
コストパフォーマンスの高さ

HarborS表参道では住所利用と来客対応を基本料金に含まれるサービスとして月額4800円で提供しております。バーチャルオフィスの中には格安を謳い、1000円未満やワンコインでの住所利用を提供している事業者もあり、そういった業者と比較すると少し割高に感じてしまうかもしれませんが、提供できる住所の価値を考えると月額4800円という価格はバーチャルオフィスとしてはお手頃な価格と言えます。

また、バーチャルオフィスを利用せずに「港区南青山」でオフィスを借りようとする場合と比較すると、価格の安さをより分かりやすく想像していただけると思います。例えば港区で賃貸オフィスを借りるとすると、物件の広さなどにもよりますが、最低でも1坪当たり18000円はかかってしまいます。たった1坪で、HarborSのバーチャルオフィスのオプションを全て付けてもお釣りが来るような高さです。オフィスとして活用するには当然1坪では意味が無いので、必要最低限の広さでも賃料は大きな負担になります。

都内一等地の土地による付加価値をはじめ、「イメージ」「価格」「利便性」のバランスで見た時のコストパフォーマンスの高さこそが、多くのお客様にご利用いただけている所以なのだと感じています。

最後に

HarborS表参道では月額4800円からバーチャルオフィスを利用できます。港区南青山というブランド力の高い住所をこの料金で利用できて、さらに駅から近いバーチャルオフィスは他にありません。
今後は拠点展開も見据えており、よりお客様にとって利用しやすいバーチャルオフィス運営を目指してまいります。
