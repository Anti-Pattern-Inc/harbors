特定商取引法について

インターネットで会社を検索すれば、ほとんどの会社はホームページに住所・氏名、電話番号などを掲載しています。しかし中には、個人情報を知られたくないという理由から、これらの情報を記載しない会社もあります。その多くは個人事業主です。個人事業主の場合は、住所と電話を表記しなくても法的には何の問題もありません。

しかし、自分が消費者の立場に立った場合、果たして連絡先がわからないところのサービスを利用しようと思うでしょうか。社会的信用を得るために、またユーザーに安心してサービスを利用してもらうために、住所や電話番号は表記しておいたほうが望ましいです。特にネットショップを開設する場合には、特定商取引法により、事業者の氏名・住所・電話番号などを表記することが義務付けられています。仮に住所と電話番号などを表記しないでビジネスを続けた場合、行政処分を受けるおそれがあるため注意が必要です。

そもそも特定商取引法の目的は、消費者トラブルが生じやすい特定の取引を対象としたトラブルの防止と消費者の利益を保護することです。その目的を果たすために事業者が守るべきルールとクーリング・オフなど消費者を守る制度を設けています。

実際に特定商取引法の対象になる取引類型は下記の7種類になります。ネットショップはこのうちの『2.通信販売』にあたります。

  1. 訪問販売
  2. 通信販売
  3. 電話勧誘販売
  4. 連鎖販売取引
  5. 特定継続的役務提供
  6. 業務提供誘引販売取引
  7. 訪問購入

また、特定商取引法に設けられている事業者が守るべきルールのひとつに広告の表示義務があります。これは事業者の氏名（名称）、住所、電話番号などの個人情報を消費者に分かる形で表示しなければならないというものです。

ただし、これには例外があり、表示事項の一部を開示しなくてもよいとされています。下記のような場合は、特定商取引法の表示義務違反に該当しません。

  • 掲載している氏名や住所、電話番号などの情報が事業者のものではないことを明記する
  • ユーザーから「開示してほしい」という請求に対して、速やかに情報提供できる状態にしている

例えば、個人や女性の方がネットショップを運営する場合、自分の氏名や住所、電話番号などの個人情報を一般公開することに抵抗を感じる方も少なくないでしょう。最近ではそのような場合に、バーチャルオフィスを利用してネットショップを開設する人も増えています。

サイト上に事業者の個人情報ではなく、バーチャルオフィスの名称・住所・電話番号であるということを明記し、ユーザーから開示請求があれば速やかに公開できる準備をしている場合は、何の問題もなく運営できます。

バーチャルオフィスを利用すれば、個人情報が漏洩するリスクを回避でき、さらに初期費用や月額の賃料、毎月の出費の軽減も可能です。
