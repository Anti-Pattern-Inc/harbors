ACCESS

HarborS 表参道は、港区青山エリアに位置しており、表参道駅A4出口から徒歩3分という好立地です。
住所は「港区南青山」とブランド力抜群のこのエリアは、渋谷、原宿、麻布、赤坂、六本木エリアとも隣接しており、どこからのアクセスも抜群の立地という特徴があります。

[周辺からのアクセス]

六本木：千代田線(乃木坂駅利用)で2分 (徒歩25分圏内)

赤坂：千代田線で6分 (徒歩30分圏内)
