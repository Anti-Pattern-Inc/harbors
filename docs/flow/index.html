バーチャルオフィス ご契約までの流れ

Step1 バーチャルオフィス見学予約

見学予約フォームから、バーチャルオフィス見学の予約を行います。
お気軽に、ご見学予約をお申し込みください。

Step2 バーチャルオフィス見学

専用ポスト、カフェラウンジ、会議室等を実際にご覧頂きます。また、弊社スタッフにより、サービス内容の説明や料金体系のご説明をさせていただきます。
必要な方には、お見積もりをお出し致します。また、希望者にはその場で面談もさせて頂きます。

Step3 お申し込み

オンラインにてお申し込みが可能となります。

Step4 必要書類の送付

メールにて、必要書類を送付頂きます。
個人でご契約の場合は写真付きの身分証明書、法人の場合は履歴事項全部証明書は写真付きの身分証明書をご用意ください

Step5 面談による入居審査

弊社では、対面での面談をさせて頂きます。
ご入居にあたりましての審査を実施させて頂きます。３営業日程度のお時間を頂く可能性があります。

Step6 正式契約

必要事項を記載の上、利用規約を確認頂き、オンラインにて契約を締結致します。

Step7 支払い情報の登録

決済に利用するクレジットカード情報をご登録頂きます。
基本的には、クレジットカードでの決済をお願いしておりますが、銀行振込を希望される方は、弊社スタッフまでご相談ください。

Step8 サービス利用開始

契約が完了となり、サービスの利用の開始です。
